# LOGIN PAGE SPEC

## Layout
- Sử dụng Tabs:
  - Admin
  - Teacher
  - Parent

## Mỗi Tab gồm:
- Email
- Password
- Remember me
- Nút Đăng nhập
- Quên mật khẩu

## Yêu cầu UI
- Centered card
- Background gradient giống homepage
- PillNav hiển thị tối giản
- Animation fade-in
- Icon user theo từng role

Admin: icon shield
Teacher: icon book
Parent: icon user

## Điều hướng
- Sau login:
  Admin → /admin
  Teacher → /teacher
